## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Start Client in front-end dir:  

`npm start`  

To Start Server back-end dir: 

`npm start`  

To Visit App:

`localhost:3000/`  

Compose with Docker:

`docker-compose up`
